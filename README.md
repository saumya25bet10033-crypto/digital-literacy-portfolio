## Project Overview
This project was created as part of the Digital Literacy course. The purpose of this portfolio is to demonstrate basic digital skills required for students such as creating professional profiles, understanding cyber safety, professional communication and using online collaboration tools.

Through this project I learned how to create a professional digital presence, how to communicate through proper email etiquette and how to stay safe from cybercrime. This project also helped me understand how platforms like GitHub and HackerRank can help in improving technical skills.

## Task Summary

### Task 1 – Digital Literacy Infographic
I created an infographic using Canva explaining digital literacy, useful digital tools and safe internet practices.

### Task 2 – Digital Portfolio
I created my professional profiles on GitHub, LinkedIn and HackerRank to start building my digital presence.

### Task 3 – Platforms
I solved some beginner coding problems on HackerRank and created a Google Form quiz related to digital literacy awareness.

### Task 4 – Email Etiquette
I drafted two professional emails and created a social media responsibility checklist.

### Task 5 – Cybercrime Awareness
I prepared a case study on phishing and created a prevention checklist for students.
